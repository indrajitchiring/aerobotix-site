Setup Instructions

Aerobotix Enrollment Form → Google Sheets Integration · One-time setup · ~10 minutes

Step 1

Create a Google Sheet

Go to sheets.google.com and create a new blank spreadsheet.

Name it something like "Aerobotix Enrollments". Leave it empty — the script will auto-create the headers on the first submission.

Step 2

Open Apps Script Editor

Inside your Google Sheet, click the menu: Extensions → Apps Script

This opens the Apps Script editor in a new tab. Delete any default code you see in the editor (the empty myFunction() block).

Step 3

Paste the Script (Code.gs)

Open the file Code.gs (provided alongside this guide). Copy its entire contents and paste it into the Apps Script editor.

Press Ctrl + S (or click the Save icon) to save. Name your project "Aerobotix Form Handler" when prompted.

Step 4

Deploy as a Web App

In the Apps Script editor, click the blue "Deploy" button (top right) → "New deployment".

  • Type: Select "Web app"
  • Description: Aerobotix Enrollment Form
  • Execute as: Me (your Google account)
  • Who has access: Anyone ← This is important so the form can submit

Click Deploy. Google will ask you to Authorize the script — click through and allow access.

✅ After deploying, you'll see a Web App URL that looks like:
https://script.google.com/macros/s/AKfycb.../exec
Copy this URL — you need it in Step 5.
Step 5

Paste URL into the HTML Form

Open aerobotix_enrollment_form.html in any text editor (Notepad, VS Code, etc.).

Find this line near the bottom of the file:

const APPS_SCRIPT_URL = "YOUR_APPS_SCRIPT_URL_HERE";

Replace YOUR_APPS_SCRIPT_URL_HERE with the Web App URL you copied in Step 4. Keep the quotes.

⚠️ Every time you make a change to Code.gs and redeploy, Google creates a new version. Make sure you always click "Manage deployments" → edit existing deployment → "New version" rather than creating a fresh deployment, so the URL stays the same.
Step 6

Host the HTML Form (Choose One)

The HTML file is self-contained. You can host it anywhere:

  • Simplest: Upload to GitHub Pages (free, instant HTTPS)
  • Your website: Upload via cPanel or FTP to any web host
  • Quick share: Use tiiny.host (drag & drop, free)
  • Offline/local: Just open the .html file in a browser — still works if internet is available for the form submission
Step 7

Test It

Open the form, fill in all fields, and submit. Then check your Google Sheet — a new row should appear within a few seconds with all the student's information.

The sheet columns will be: Timestamp · Full Name · DOB · Gender · Address · Mobile · WhatsApp · Email · Guardian · Qualification · Stream · Institution · Year · Occupation · Course/Track · Batch · Source · Motivation

✅ That's it! Every new submission now lands directly in your Google Sheet automatically — no database, no maintenance needed.
