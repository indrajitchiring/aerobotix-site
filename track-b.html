Track B Explorer Syllabus

How Explorer Progresses

Explorer is the foundation stage for senior learners. It moves from logic and coding into sensors, Arduino, R-Star to Challenger robotics, Bluetooth control, and hands-on manual drone operation.

Programming Foundation

Scratch, App Inventor, HTML and Python Basics

Students begin with logic and software confidence through block coding, app creation, basic web structure, and Python fundamentals so they can later control hardware and build intelligent systems.

Scratch App Inventor HTML Basics Python Basics
Electronics Foundation

Basic Electronics, Arduino and Sensor Integration

Learners build comfort with circuits, actuators, microcontrollers, and real sensor input. This is the stage where code starts interacting with the physical world in a reliable way.

Basic Electronics Arduino Sensors Motor Control
Robotics Progression

R-Star, Champion Lite, Champion and Challenger

Explorer covers the full build-and-control progression from R-Star through Challenger. Students assemble, troubleshoot, test robot behavior, and develop confidence with progressively smarter machines.

R-Star Champion Lite Champion Challenger
Control and Flight

Bluetooth Robotics, Manual Drone Flying and AI Foundations

The final Explorer stage adds phone-commanded robots, manual drone practice, and foundational AI concepts so students understand the difference between control logic and intelligent decision-making.

Bluetooth Control Manual Drone Foundational AI
Builds and Labs

What Students Build in Explorer

Explorer stays practical from start to finish. Students wire, code, assemble, test, and pilot rather than only reading concepts.

Robot Projects

Core Robotics and Control Builds

  • R-Star bridge projects such as RGB LED control, touch switch, IR comparator, automatic light controller, and H-bridge motor driver circuits.
  • Champion Lite and Champion projects using sound, light, motor, buzzer, and bumper-switch logic.
  • Autonomous behavior builds like line follower, obstacle avoider, and edge avoider systems.
  • Challenger builds including Bluetooth robot, voice control robot, gesture control robot, and gripper assembly.
Applied Practice

Electronics, Apps and Drone Readiness

  • Arduino wiring and sensor experiments using real boards, breadboards, and serial programming.
  • Simple app-based control interfaces through App Inventor for real robot interaction.
  • Bluetooth command workflows that let robots respond to mobile phone inputs.
  • Manual drone setup, controller handling, safe lift-off, landing, and flight discipline.
